# 📊 Data Collection & Management Project

## 📌 Overview
This project analyzes a real-world sales dataset to demonstrate key concepts in data collection, data management, and data analysis. The dataset is processed using both Excel and Python to extract insights and create meaningful visualizations.

---

## 🎯 Objectives
- Understand the data lifecycle and governance
- Work with structured data from CSV files
- Clean and preprocess raw data
- Perform exploratory data analysis (EDA)
- Generate business insights using Excel
- Calculate KPIs using Python (Pandas)

---

## 📂 Dataset
- File: `Sales-Data.csv`
- Type: Structured data (tabular format with rows and columns)
- Format: CSV (Comma-Separated Values)

---

## 🛠️ Tools & Technologies
- Microsoft Excel (Pivot Tables, Charts, Functions)
- Python (Pandas, Jupyter Notebook)
- CSV file format

---

## 📊 Project Tasks

### 🔹 Part A: Excel Analysis
- Imported and explored dataset
- Identified column purposes
- Calculated:
  - Total revenue by product
  - Revenue by region
  - Top 3 best-selling products
  - Monthly revenue trends
- Created visualizations (bar/column charts)

---

### 🔹 Part B: Data Cleaning (Python)
- Loaded dataset using Pandas
- Checked for:
  - Missing values
  - Duplicates
  - Incorrect data types
- Cleaned dataset:
  - Removed duplicates
  - Handled missing values
  - Converted date columns to datetime format

---

### 🔹 Part C: KPI Analysis (Python)
Calculated key performance indicators:
- Total revenue
- Average units sold per order
- Revenue by region
- Highest-performing sales representative
- Top 3 products by units sold

---

## 📈 Key Insights
- Identified top-performing products and regions
- Analyzed monthly sales trends
- Evaluated sales representative performance
- Generated actionable business insights
